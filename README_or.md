ଯେକୌଣସି ଆଣ୍ଡ୍ରଏଡ୍ ୯–୧୬ ଡିଭାଇସରେ ସର୍କଲ୍ ଟୁ ସର୍ଚ୍ଚ ସକ୍ରିୟ କରନ୍ତୁ

*ଏହି ଆପ୍ କେବଳ ସର୍କଲ୍ ଟୁ ସର୍ଚ୍ଚକୁ ଆରମ୍ଭ କରିବା ପାଇଁ ତିଆରି କରାଯାଇଛି ଏବଂ ସଫଳତାପୂର୍ବକ ଆରମ୍ଭ ହେବା ପରେ ଯଦି କୌଣସି ସମସ୍ୟା ଉପୁଜେ, ତେବେ ଏହା ତାହା ସମାଧାନ କରିପାରିବ ନାହିଁ*

## ବ୍ୟବହାର କରିବାର ପ୍ରଣାଳୀ

୧. [Google](https://play.google.com/store/apps/details?id=com.google.android.googlequicksearchbox) ଆପର ସର୍ବଶେଷ ସଂସ୍କରଣ ସ୍ଥାପନ କରନ୍ତୁ, ଅଟୋ-ଷ୍ଟାର୍ଟ (auto-start) ଅନୁମତି ଦିଅନ୍ତୁ, ବ୍ୟାକଗ୍ରାଉଣ୍ଡ କଟକଣା ବନ୍ଦ କରନ୍ତୁ ଏବଂ ଗୁଗଲ୍‌କୁ ଡିଫଲ୍ଟ ଆସିଷ୍ଟାଣ୍ଟ ଆପ୍ ଭାବରେ ସେଟ୍ କରନ୍ତୁ।

୨. MiCTS ସ୍ଥାପନ କରନ୍ତୁ ଏବଂ ଲଞ୍ଚ୍ କରନ୍ତୁ
   - ଯଦି ଆପଣଙ୍କ ଭାଗ୍ୟ ଭଲ ଥାଏ, ତେବେ MiCTS ଲଞ୍ଚ୍ କରିବା ସମୟରେ ବିନା ରୁଟ୍‌ରେ ସର୍କଲ୍ ଟୁ ସର୍ଚ୍ଚ ସକ୍ରିୟ ହୋଇଯିବ।
   - ଯଦି କିଛି ହେଉନାହିଁ, ତେବେ ବୋଧହୁଏ ଗୁଗଲ୍ ଆପଣଙ୍କ ଡିଭାଇସ୍ ପାଇଁ ଏହାକୁ ବନ୍ଦ କରି ରଖିଛି (ଆପଣ Logcat ରେ `Omni invocation failed: not enabled` ମେସେଜ୍ ଯାଞ୍ଚ କରି ଏହା ନିଶ୍ଚିତ କରିପାରିବେ)। **ରୁଟ୍ ସହିତ** ନିମ୍ନଲିଖିତ ପ୍ରୟାସ କରନ୍ତୁ:
     - LSPosed ରେ ମଡ୍ୟୁଲ୍‌ଟିକୁ ଆକ୍ଟିଭେଟ୍ କରନ୍ତୁ, [MiCTS settings](#settings) ରେ `Device spoof for Google` ଚାଲୁ କରନ୍ତୁ ଏବଂ ଗୁଗଲ୍ ଆପ୍‌କୁ ଫୋର୍ସ ଷ୍ଟପ୍ କରି ପୁଣି ଖୋଲନ୍ତୁ।
     - ଯଦି ତଥାପି କାମ କରୁନାହିଁ, ତେବେ [GMS-Flags](https://github.com/polodarb/GMS-Flags) ବ୍ୟବହାର କରି `com.google.android.apps.search.omnient.device` ଫ୍ଲାଗ୍ `45631784` କୁ true କରନ୍ତୁ।

୩. ଟ୍ରିଗର୍ ପଦ୍ଧତି ସେଟ୍ ଅପ୍ କରନ୍ତୁ
   - MiCTS ଖୋଲିବା ମାତ୍ରେ ଏହା ଟ୍ରିଗର୍ ହେବ, ତେଣୁ ଆପଣ Quick Ball, Xposed Edge, ShortX ଇତ୍ୟାଦି ଆପ୍ ବ୍ୟବହାର କରି MiCTS ଲଞ୍ଚ୍ କରିବାକୁ ଏକ ଆକ୍ସନ ଭାବେ ସେଟ୍ କରିପାରିବେ।
   - MiCTS ଏକ ଟ୍ରିଗର୍ ଟାଇଲ୍ ପ୍ରଦାନ କରେ, ଯାହାକୁ ଆପଣ କୁଇକ୍ ସେଟିଂସ ପ୍ୟାନେଲରେ ଯୋଡ଼ି କ୍ଲିକ୍ କରି ଟ୍ରିଗର୍ କରିପାରିବେ।
   - Xiaomi ଡିଭାଇସ୍ ପାଇଁ, MiCTS ରେ `Trigger by long press gesture handle` ଏବଂ `Trigger by long press home button` ପାଇଁ ବିଲ୍ଟ-ଇନ୍ ସପୋର୍ଟ ଅଛି (ଏଥିପାଇଁ ମଡ୍ୟୁଲ୍ ଆକ୍ଟିଭେଟ୍ କରିବା ଏବଂ ଫୋନ୍ ରିଷ୍ଟାର୍ଟ କରିବା ଆବଶ୍ୟକ)।
   - Android 13 ଏବଂ ତା'ଠାରୁ ଅଧିକ ସଂସ୍କରଣର Samsung ଡିଭାଇସ୍ ପାଇଁ, [Galaxy Store](https://galaxystore.samsung.com/detail/com.samsung.android.app.routineplus) କିମ୍ବା [Good Lock](https://galaxystore.samsung.com/detail/com.samsung.android.goodlock) ରୁ "Routines+" ସ୍ଥାପନ କରନ୍ତୁ। ତା'ପରେ ପାୱାର ବଟନ୍‌କୁ ଦବାଇ ରଖିଲେ MiCTS ଖୋଲିବା ପାଇଁ ରୁଟିନ୍ ତିଆରି କରନ୍ତୁ।

## ସେଟିଂସ୍

### ସେଟିଂସ ଭିତରକୁ କିପରି ଯିବେ
- MiCTS ଆପ୍ ଆଇକନ୍‌କୁ କିଛି ସମୟ ଦବାଇ ରଖନ୍ତୁ ଏବଂ Settings ବିକଳ୍ପ ଉପରେ କ୍ଲିକ୍ କରନ୍ତୁ।
- LSPosed ର Modules ପୃଷ୍ଠାରୁ MiCTS କୁ ଯାଇ ସେଟିଂସ ଆଇକନ୍ କ୍ଲିକ୍ କରନ୍ତୁ।
- କୁଇକ୍ ସେଟିଂସ ପ୍ୟାନେଲ୍ ଟାଇଲ୍‌କୁ କିଛି ସମୟ ଦବାଇ ରଖନ୍ତୁ।

### ଆପ୍ ସେଟିଂସ୍
- Default trigger delay: MiCTS ଲଞ୍ଚ୍ କରି ଟ୍ରିଗର୍ କରିବା ସମୟରେ ହେଉଥିବା ବିଳମ୍ବ।
- Tile trigger delay: କୁଇକ୍ ସେଟିଂସ ଟାଇଲ୍ ମାଧ୍ୟମରେ ଟ୍ରିଗର୍ କରିବା ସମୟରେ ହେଉଥିବା ବିଳମ୍ବ।

### ମଡ୍ୟୁଲ୍ ସେଟିଂସ୍
LSPosed ରେ ମଡ୍ୟୁଲ୍ ଆକ୍ଟିଭେଟ୍ କରିବା ଆବଶ୍ୟକ
- System trigger service: ଟ୍ରିଗର୍ ପାଇଁ ବ୍ୟବହୃତ ସିଷ୍ଟମ୍ ସର୍ଭିସ୍। କେବଳ ସପୋର୍ଟେଡ୍ ସର୍ଭିସ୍ ଦେଖାଯିବ। LSPosed ର ସ୍କୋପ୍‌ରେ System Framework ଯୋଡ଼ିବା ଆବଶ୍ୟକ।
   - VIS: Android 9-16 ରେ ସପୋର୍ଟ କରେ। ଗୁଗଲ୍‌କୁ ଡିଫଲ୍ଟ ଆସିଷ୍ଟାଣ୍ଟ ଭାବେ ସେଟ୍ କରିବାକୁ ପଡ଼ିବ।
   - CSHelper: Android 14 QPR3 ଏବଂ ତା'ଠାରୁ ଅଧିକରେ ସପୋର୍ଟ କରେ। ଗୁଗଲ୍‌କୁ ଡିଫଲ୍ଟ ଆସିଷ୍ଟାଣ୍ଟ ଭାବେ ସେଟ୍ କରିବା ଜରୁରୀ ନୁହେଁ।
   - CSService: Android 15 ଏବଂ ତା'ଠାରୁ ଅଧିକରେ ସପୋର୍ଟ କରେ। ଏହା ସର୍କଲ୍ ଟୁ ସର୍ଚ୍ଚ ପାଇଁ ଏକ ସ୍ୱତନ୍ତ୍ର ସର୍ଭିସ୍।

- Trigger by long press gesture handle: କେବଳ Xiaomi ଡିଭାଇସ୍। LSPosed ସ୍କୋପ୍‌ରେ System Launcher/POCO Launcher ଯୋଡ଼ିବା ଆବଶ୍ୟକ।
- Trigger by long press home button: କେବଳ Xiaomi ଡିଭାଇସ୍। LSPosed ସ୍କୋପ୍‌ରେ System Framework ଯୋଡ଼ିବା ଆବଶ୍ୟକ।
- Device spoof for Google: LSPosed ସ୍କୋପ୍‌ରେ Google ଯୋଡ଼ିବା ଆବଶ୍ୟକ। Manufacturer, Brand, Model ଏବଂ Device ର ମୂଲ୍ୟ ବଦଳାଇବାକୁ ଏହା ବ୍ୟବହୃତ ହୁଏ।

## ସାଧାରଣ ପ୍ରଶ୍ନ

### "Trigger failed!" ପ୍ରମ୍ପ୍ଟ ଦେଖାଉଛି?
ବୋଧହୁଏ ଗୁଗଲ୍ ଡିଫଲ୍ଟ ଆସିଷ୍ଟାଣ୍ଟ ଭାବେ ସେଟ୍ ହୋଇନାହିଁ, ଏହାକୁ ଯାଞ୍ଚ କରନ୍ତୁ।

### ସର୍କଲ୍ ଟୁ ସର୍ଚ୍ଚ ବଦଳରେ ଗୁଗଲ୍ ଆସିଷ୍ଟାଣ୍ଟ ଆସୁଛି?
ନିଶ୍ଚିତ କରନ୍ତୁ ଯେ ଗୁଗଲ୍ ଆପ୍ ସର୍ବଶେଷ ସଂସ୍କରଣରେ ଅଛି।

### ବେଳେବେଳେ ସଫଳତାର ସହ ଟ୍ରିଗର୍ ହେଉନାହିଁ?
ଏହା 'tombstone' ମେକାନିଜମ୍ ଯୋଗୁଁ ହୋଇଥାଇପାରେ। ବ୍ୟାଟେରୀ ସେଭର୍ ସେଟିଂସରେ ଗୁଗଲ୍ ପାଇଁ "No restrictions" ବାଛନ୍ତୁ।
